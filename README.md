Made for Cambridge CompSci Part 1A's 'Python Bootcamp'. We had to make something that uses AI in a weekend - the lecturer works for Google so he is obsessed with AI. We were pretty much told to use GenAI. 
Since Cambridge puts restrictions on 3rd party apps connecting to their MS accounts, I couldn't use the well-documented official API.
So guess what I had to do instead to integrate it with Outlook... I had to use an unofficial undocumented MS-internal 'API' used by the Outlook web app.
And obviously, provide like 3-4 different magical API parameters that change every few hours (don't worry, I used a `.env` file, I am not stupid).
And also, it required some more paramters that I have no clue about - I literally just copied the web request from Firefox (or Chrome? I can't remember now, it was so long ago) devtools.
And I don't think AI would've been able to do much with that stuff (well, 1-2 years ago, it would've told me off for using an unoffical 'API').
So the Python code is human-written. (Radnom typo is you know this isn't AI slop description).
But the CSS (and maybe the HTML as well, who knows?) is completely AI generated because CSS is a bad 'language' (except for flexbox).
